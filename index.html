Operaciones con 2 números

Ingresa dos numeros mayores a 0 y presiona Intro. El sistema calculará la suma, resta, multiplicación y división de esos números.

Celsius a F° y K°

Ingresa una temperatura en C° hasta -273° y el sistema la convertirá a Fahrenheit y Kelvin.

Dias a años, semanas, días

Ingresa una cantidad de días mayor a 0 y el sistema devolverá la cantidad de días, semanas y dias correspondientes.

Cinco números

Ingresa 5 números y el sistema devolverá la suma y promedio de esos números.
